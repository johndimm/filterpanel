# filterpanel

This is a react component that generates a filter panel from a table of data.   Filter panels are used by Amazon, Walmart, and many other e-commerce sites.    

## Installation:

```
  npm install filterpanel --save
```

Publish steps:

```
  npm run build
  npm publish
```
